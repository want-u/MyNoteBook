# 自适应单位
https://www.cnblogs.com/0506winds/p/13953600.html

```
def format_size(value):
    # 自适应转化单位KB、MB、GB
    units = ["Byte", "Kb", "Mb", "Gb", "Tb", "Pb"]
    size = 1024.0
    for i in range(len(units)):
        if (value / size) < 1:
            return "%.2f %s" % (value, units[i])
        value = value / size
```
